## Zava DIY를 위한 합성 데이터 생성

Zava DIY는 개발자가 테스트 및 개발 목적으로 합성 데이터를 생성할 수 있도록 도와주는 도구입니다. 사용자가 프라이버시나 보안을 침해하지 않으면서 특정 요구사항을 충족하는 현실적인 데이터셋을 생성하여 다양한 애플리케이션에서 사용할 수 있도록 합니다.

[Zava DIY 제품 데이터셋](https://github.com/microsoft/ai-tour-26-zava-diy-dataset-plus-mcp/tree/main){:target="_blank"} GitHub 저장소.

데이터베이스에 포함된 내용:

- **워싱턴 주 전역의 8개 매장**, 각각 고유한 재고 및 판매 데이터 보유
- **50,000개 이상의 고객 기록** (워싱턴 주 전역 및 온라인)
- **400개 이상의 DIY 제품** (도구, 야외 장비, 주택 개선 용품 포함)
- **400개 이상의 제품 이미지** (이미지 기반 검색을 위해 데이터베이스에 연결)
- **200,000개 이상의 주문 거래** (상세한 판매 이력 포함)
- **3,000개 이상의 재고 항목** (여러 매장에 걸쳐)
- **이미지 임베딩** ([openai/clip-vit-base-patch32](https://huggingface.co/openai/clip-vit-base-patch32/blob/main/README.md){:target="_blank"}를 사용하여 인코딩)으로 AI 기반 유사성 검색 가능
- **텍스트 임베딩** ([openai/text-embedding-3-small](https://ai.azure.com/catalog/models/text-embedding-3-small){:target="_blank"})으로 제품 설명을 통한 검색 및 추천 기능 향상

데이터베이스는 복잡한 쿼리와 분석을 지원하여 판매, 재고, 고객 데이터에 효율적으로 액세스할 수 있습니다. PostgreSQL 행 수준 보안(RLS)은 에이전트가 할당된 매장의 데이터에만 액세스할 수 있도록 제한하여 보안과 프라이버시를 보장합니다.
