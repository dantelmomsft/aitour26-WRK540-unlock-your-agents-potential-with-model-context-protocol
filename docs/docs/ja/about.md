## Zava DIY の合成データ生成

Zava DIY は、開発者がテストや開発目的で合成データを作成するのを支援するために設計されたツールです。これにより、ユーザーは様々なアプリケーションで使用できる現実的なデータセットを生成でき、プライバシーやセキュリティを損なうことなく特定の要件を満たすデータを確保できます。

[Zava DIY 製品データセット](https://github.com/microsoft/ai-tour-26-zava-diy-dataset-plus-mcp/tree/main){:target="_blank"} GitHub リポジトリ。

データベースには以下が含まれます：

- **8店舗** ワシントン州全域に、それぞれ独自の在庫と売上データを持つ
- **50,000以上の顧客記録** ワシントン州全域とオンライン
- **400以上のDIY製品** 工具、アウトドア設備、住宅改良用品を含む
- **400以上の製品画像** 画像ベースの検索のためにデータベースとリンク
- **200,000以上の注文取引** 詳細な売上履歴付き
- **3,000以上の在庫アイテム** 複数店舗にわたって
- **画像埋め込み** 製品画像用で、AI による類似検索を可能にする（[openai/clip-vit-base-patch32](https://huggingface.co/openai/clip-vit-base-patch32/blob/main/README.md){:target="_blank"} を使用してエンコード）
- **テキスト埋め込み** 製品説明用で、検索と推薦機能を強化 [openai/text-embedding-3-small](https://ai.azure.com/catalog/models/text-embedding-3-small){:target="_blank"}

データベースは複雑なクエリと分析をサポートし、売上、在庫、顧客データへの効率的なアクセスを可能にします。PostgreSQL 行レベルセキュリティ（RLS）により、エージェントは割り当てられた店舗のデータのみにアクセスが制限され、セキュリティとプライバシーが確保されます。
